![GitHub License](https://img.shields.io/github/license/shenke0721/MZProxy-Master)

## 🚀 简介
全网最强代理中转管理脚本，支持全协议、BBR加速、中转转发等功能。

## 📦 一键安装
bash

bash <(curl -sSL https://raw.githubusercontent.com/shenke0721/MZProxy-Master/main/install.sh )

## ✨ 功能特性
- 支持 VLESS/VMess/Trojan/Shadowsocks 等全协议
- BBR 加速网络优化
- GOST 中转转发
- 自动申请 SSL 证书
- Web 面板管理

## 📄 使用说明
1. 确保服务器是 root 权限
2. 执行一键安装命令
3. 按照菜单提示操作
